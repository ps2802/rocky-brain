# Recovery Procedure (Ops View)

1. Restore the VPS or provision a fresh one.
2. Install Hermes and required system dependencies.
3. Clone or pull `rocky-brain` to its stable path.
4. Install `SOUL.md` into `~/.hermes/SOUL.md`.
5. Re-expose `skills/` to Hermes.
6. Launch Hermes in this repo context and follow `RECOVERY.md`.
7. Verify `runtime/tasks-work.md`, `runtime/tasks-personal.md`, and the latest relevant memory files are present.
8. Log the recovery result in `logs/daily-log.md` or `logs/session-log.md`.
