# Server Setup

1. Provision VPS access for the intended operator.
2. Install Hermes and required system dependencies.
3. Clone or pull `rocky-brain` to a stable path such as `~/rocky-brain`.
4. Install `SOUL.md` into `~/.hermes/SOUL.md`.
5. Expose this repo's `skills/` directory to Hermes.
6. Configure Telegram and provider credentials in `~/.hermes/.env`.
7. Verify Hermes can read this repo as project context and list the installed Rocky skills.
