# Server Setup

1. Provision VPS (Ubuntu 22.04+ recommended) with access for Praneet + Rocky.
2. Install dependencies:
   - Node.js 22+
   - Git
   - OpenClaw runtime + required services
3. Clone/pull `rocky-brain` repo into `~/.openclaw/workspace` (or desired path).
4. Configure Telegram/OpenClaw integration per OpenClaw docs.
5. Ensure regular backups of the workspace + logs.
