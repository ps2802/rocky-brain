# RECOVERY.md - How to Restore Rocky

1. **Clone Repo**
   - `git clone git@github.com:ps2802/rocky-brain.git ~/.openclaw/workspace` (once remote exists) or copy files into workspace.
2. **Load Anchors**
   - Read SOUL.md, USER.md, STACK.md, MISSION.md, BRAIN.md, MEMORY.md.
3. **Check Logs**
   - Review `logs/daily-log.md`, `logs/session-log.md`, and latest `memory/YYYY-MM-DD.md` for current state.
4. **Task Sync**
   - Open TASKS.md for active queues; cross-reference project files.
5. **Tooling Check**
   - Run `openclaw status`, confirm Telegram + model access.
6. **Resume Work**
   - Continue executing tasks, updating files, and logging changes.
7. **Document Recovery**
   - Add entry to `logs/session-log.md` + current daily log.
