# Skills

This directory contains Rocky-specific Hermes skills.

## Install

Choose one:

1. Copy each `rocky-*` skill directory into `~/.hermes/skills/`
2. Add this repo's `skills/` directory as an external Hermes skill path

## Included Skills

- `rocky-stratt`
- `rocky-wrench`
- `rocky-sputnik`
- `rocky-forge`
- `rocky-velvet-knife`
- `composio-agent-ops`

These skills are reusable role packets and workflows. They do not depend on a fake `/spawn` command.
